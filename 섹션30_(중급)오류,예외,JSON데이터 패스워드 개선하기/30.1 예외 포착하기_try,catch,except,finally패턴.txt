try: Something that might cause an exception = 예외의 원인이 될 만한 것
예외를 유발할수 있는 뭔가를 실행하는 코드블록을 가리킨다. 그래서 어떤 라인을 실행하려 시도하고 대부분의
경우에는 아마 작동을 할 것이지만 가끔은 그렇지 않을 수도 있다.

except: Do this if there was an exception = 예외가 있으면 이렇게 하다.
만약에 예외가 있었다면 컴퓨터가 실행하게 하련느 블록이다. 만약에 뭔가가 정말로 잘못 되었고 우리가 원한 방식이
아니라면 바로 이 코드 블록을 실행하게 된다.

else: Do this if there were n exception =  예외 없이 이것을 하다.
예외가 없었을때 실행할 코드를 정의할수 있다. 실패할수 있는 이런 것을 시도했지만 실제로 실패하지 않았고
성공해서 문제가 없을때 else 블록 안에 있는 것을 하게 될거다.

finally: Do this no matter what happens = 무슨일이 있어도 이렇게 하세요.
기본적으로 어떤 일이 일어나더라도 실행해야 할 코드블록을 가리킨다. 만약 우리가 시도한 것이 실패했거나
아니면 성공했거나, 이 코드 라인으로 시도해서 뭐가 일어나든 이 finally 블록은 항상 실행될것이다.
코드 실행 마지막에 뭔가를 지우거나 정리하기 위해 사용된다.
